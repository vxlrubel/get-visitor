# Get Visitor WordPress Plugin

## Description

The "Get Visitor" WordPress plugin is a lightweight tool designed to facilitate email subscription collection from site visitors. Leveraging the WordPress REST API, this plugin provides an easy interface for visitors to subscribe to your email list. Upon visitor subscription through the provided form, their email addresses are securely collected and stored for your email marketing purposes.

## Features

- **Simple Integration**: Seamlessly integrate an email subscription form on your WordPress site.
- **REST API**: Utilizes WordPress REST API for efficient data collection.
- **Secure Storage**: Safely stores collected email addresses for your email marketing needs.
- **Lightweight**: Minimalistic design and resource-efficient implementation.

## Installation

1. [Download](https://codeload.github.com/vxlrubel/get-visitor/zip/refs/heads/main) the plugin ZIP file.
2. Log in to your WordPress admin panel.
3. Navigate to `Plugins > Add New`.
4. Click `Upload Plugin` and select the downloaded ZIP file.
5. Activate the plugin.

## Usage

1. Access the plugin settings from the WordPress admin dashboard.
2. Customize the subscription form fields and design.
3. Place the form on your desired website pages.

## Requirements

- WordPress version 4.7 or higher.
